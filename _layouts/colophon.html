Colophon

The tamouse-theme is based on the standard Jekyll minima theme. I created it especially to use on my http://tamouse.org family of websites, at least the ones that are using Jekyll.

I am using the following font collection:

  • Eczar for the base font
  • Tillana for cursive text
  • Space Mono for monospaced text
  • Work Sans for headings and for buttons

The design is my own, although certainly remeniscient of `minima` but these things do tend to be pretty common, with an eye towards mobile-first design.
